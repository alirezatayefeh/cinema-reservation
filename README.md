A simple Java + Spring Boot project for managing movie ticket reservations.

## Overview
This project allows users to:

- View available seats for a movie
- Reserve seats safely with basic concurrency handling
- Explore a practical example of a backend system with Spring Boot

It's mainly for learning purposes and experimenting with multi-threading in Java.

## Requirements
- Java 17+
- Gradle (wrapper included)
- An IDE like IntelliJ IDEA or VSCode

## Running the Project
Clone the repository and run with Gradle Wrapper.
